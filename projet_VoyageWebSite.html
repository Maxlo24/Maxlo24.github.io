Site web
agence de voyage

L'objectif était de réaliser un prototypage de site web destiné à une agence de voyage fictive.

Dans le cadre de mes études nous avons été formés aux bases de la programmation web. Nous devions utiliser le language HTML, CSS et Java Script.
Pour mettre en application nos connaissances, une mission nous a été donnée :
réaliser un site web destiné à une agence de voyage fictive.


1

Cahier des charges

Résumé du cahier des charges.

Le site doit être responsive et doit au moins posseder les pages suivantes :
• une page principale qui présente les différentes destinations en photo
• une page de Réservation
• une page Espace client
• une page A propos & Contact

Chaque page doit comporter les éléments suivants :
• un menu de navigation
• un pied de page
• un bouton de retour au sommet de la page
